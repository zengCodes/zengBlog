# 《CSS知识》- CSS盒模型

# IE盒模型

盒子总宽度= width(包含内容width、padding、border) + margin

# w3c盒模型

盒子总宽度= 内容width + padding + border + margin

# css属性

盒子模型由box-sizing控制

  • IE盒模型：border-box
  • w3c盒模型：content-box
