# 《HTML知识》- HTML内联

# 一、iframe是什么

相当于一个容器，可以显示其他文档内容

# 二、为什么要使用iframe

  • iframe能够原封不动地显示文档内容
  • 如果iframe在多个网页共用，只需要修改iframe内容就可以，达到组件高可用的效果

# 三、iframe的缺点

  • 会阻塞主页面的onload事件
  • 搜索引擎不会检索内联内容，不利于SEO
  • 页面在同一区域会共享连接，影响页面加载速度
