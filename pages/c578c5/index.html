# LESS

# 解决的痛点

  • 本质是为了写更少的代码，提高开发效率
  • 增加混合、变量、函数等语法，可以提高代码扩展性和复用性，比如一键换肤或者修改网站主题等
  • 在项目复杂程度更高的情况下变得可维护

# 语法特点

  • 支持变量
  • 混合
    • 普通混合
    • 带参数混合
    • 带参数且有默认值
  • 可以引用其他标签的属性，类名（）
  • 层级嵌套，使代码更简洁
  • 增加计算符
  • @规则嵌套和冒泡
  • 可以使用内置函数
